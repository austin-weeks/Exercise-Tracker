Exercise tracker

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example: GET /api/users/12345/logs?from=2020-03-01&to=2024-08-01&limit=7
